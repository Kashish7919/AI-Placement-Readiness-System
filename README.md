# AI-Based Student Placement Readiness & Employability Analysis System

## Overview

This project predicts student placement probability using Machine Learning techniques and also analyzes placement readiness based on academic performance, employability skills, and practical exposure.

## Features

- Placement Prediction
- Placement Probability Analysis
- Feature Engineering
- Placement Readiness Score
- Recommendation System
- Feature Importance Analysis

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Machine Learning Models

- Logistic Regression
- Decision Tree
- Random Forest

## Future Improvements

- Streamlit Web App Deployment
- Deep Learning Integration
- Resume Analysis
- Interview Performance Prediction
